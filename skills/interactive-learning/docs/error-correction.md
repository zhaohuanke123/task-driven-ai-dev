# 用户纠错机制

> **本文档说明用户如何对课程内容纠错，以及AI如何处理纠错。**
>
> **AI读取时机：** 用户说"纠错X"、"这里有问题"、"不对吧"时

---

## 核心原则

**承认AI会犯错，把纠错当作学习的一部分。**

AI在生成课程时可能会"编"细节——形式正确但具体内容有误。这是AI的概率预测本质决定的。

---

## 触发条件

用户说：
- "这里有问题"
- "纠错X"
- "不对吧"
- "为什么X是Y？"

---

## 处理流程

```
用户指出错误
    │
    ├── AI承认错误，不找借口
    │
    ├── 解释正确内容
    │
    ├── 修正课程文件
    │
    └── 记录到进度.md的误区记录
```

---

## AI回应模板

```markdown
你说得对！我写错了。

正确的说法是：[正确内容]

感谢指出！这本身就是费曼学习法的精髓——发现和修正错误。
已修正课程文件，继续学习吧！
```

---

## 误区记录格式

记录到 `进度.md`：

```markdown
## 误区记录

- 日期：2024-01-16
  - 发现者：用户
  - 错误内容：美股/A股颜色含义写反
  - 纠正：美股绿色=涨，红色=跌；A股红色=涨，绿色=跌
  - 已修正文件：[[06_如何看懂行情]]
```

---

## 实战示例

### 示例：颜色含义写反

**用户**：这里的表述是不是有问题？为什么第一次看美股的人经常惊吓，红色在美股不是下跌吗？

**AI**：
```
你说得对！我写反了。

正确的逻辑是：
- A股：红色=涨，绿色=跌
- 美股：绿色=涨，红色=跌

所以"惊吓"的场景应该是：
第一次看美股的A股投资者，看到绿色满屏，以为是暴跌（因为A股绿色=跌），其实那是大涨！

感谢你仔细阅读并指出！已修正课程文件。
```

**记录到进度.md**：
```markdown
- 日期：2024-01-16
  - 发现者：用户
  - 错误内容：美股/A股颜色含义写反
  - 纠正：美股绿色=涨，红色=跌
  - 已修正文件：[[06_如何看懂行情]]
```

---

## 为什么不增加AI自我验证

1. **同一个模型验证自己不可靠**——AI"自我验证"本身也会犯错
2. **增加复杂流程但没有从根本上解决问题**
3. **用户纠错本身就是深度学习**——符合费曼学习法的精神
